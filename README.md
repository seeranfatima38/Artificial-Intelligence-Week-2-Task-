This project is a basic rule-based chatbot built using Python. It uses simple pattern matching (if-else conditions) to understand user input and respond accordingly.

The chatbot can handle common interactions like greetings, help requests, and small talk. It also includes a small knowledge base that allows it to answer specific questions related to topics like Python, AI, and chatbots.

The program runs in the console, making it interactive, and it also stores the conversation history in a text file for future reference.

# Artificial-Intelligence-Week-2-Task-
This is a beginner-level chatbot created as part of an academic project to understand the basics of conversational AI and rule-based systems.
